# Grievance Redressal System

## About
A web-based complaint management portal where students can 
submit grievances online and admins can track, assign, 
and resolve them efficiently.

Built during Summer Training at Softpro India Computer 
Technologies Pvt. Ltd. (July – September 2023)

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: PHP
- Database: MySQL
- Server: WAMP Server
- Editor: VS Code

## Key Features
- Student Registration, Login & Profile Management
- Submit, Track & Close Complaints
- Upload Picture as Proof
- Admin Dashboard for Complaint Management
- User Management & Blocked User Control
- Discussion Forum
- Session & Password Management
